# Progress Reports

This folder contains public-safe Markdown copies of the CS6727 progress reports
submitted during the Secure Model Delegation practicum project.

The reports are included so reviewers can see how the project evolved from
problem framing to prototype implementation and evaluation planning. They are
not required to run the prototype.

## Reports

| Report | Main focus |
| --- | --- |
| [Progress Report 1](progress-report-1.md) | Revised proposal, deployment plan, policy choice, benchmark plan, and evaluation plan. |
| [Progress Report 2](progress-report-2.md) | Architecture, policy-bounded controller framing, related work, benchmark schema, and methodology. |
| [Progress Report 3](progress-report-3.md) | Prototype skeleton, web UI, simulated external endpoint, policy YAML, audit logs, and early evaluation evidence. |
| [Progress Report 4](progress-report-4.md) | Expanded benchmark, baseline comparison, utility and latency analysis, and final-report planning. |

## Public-Safe Handling

These copies are provided for project evaluation context. Course-private
attachments, raw runtime artifacts, local absolute paths, and company-specific
identifiers are not included. The benchmark and examples discussed in these
reports are synthetic.
